Hi there!

It's your Address Book!
You can add new contact with command 'Add'.
Print name, phone: 'phone here',birthday: 'birthday', email: 'email', address: 'address'

 For remove contact print 'Remove' and name of contact.
 For change contact information  - print 'change' and name of contact.
 Forgot contact full name? Just print 'find' and any letters of name
 If you like to congratulate people - use command 'upcoming birth' and amount of days
 ex. if you want to see who got a birthday on this week use 7

 For all contacts use command 'show all'


For help - print "help"
For exit print -"return"
